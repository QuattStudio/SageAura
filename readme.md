# SageAura
A lightweight game engine written in C with OpenGL 3.3 Core Profile.

> ⚠️ Currently in early development (v0.3.0)

## Features
- Simple texture loading
- Window handling
- Event handling in 3 ways:
  1. Direct poll events
  2. Continuous event handling
  3. Poll events with callbacks
- 24 built-in colors, accessible via:
  - `SA_COLOR_RED` style
  - `SA_RED` style
